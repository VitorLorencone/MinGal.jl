Geometric Operations

Here you will find the main functions implemented by MinGal that you can use in your programs. For each function shown, there is a hyperlink allowing you to access the formal documentation of the function.

Products

Geometric Product

The geometric product it is the main product used when working with geometric algebra. Regarding its meaning, as noted by (DIJKMAN): "The definition is quite abstract. It is the most faithful product of oriented subspaces that one can adopt in order to form a reasonable algebra.". It can be written in terms of the inner and outer products as:

\[AB = A \cdot B + A \wedge B\]

It is applied between Blades and Multivectors and can be computed using the function geometric_product or with the * operator.

julia> Algebra(3)

julia> e2 * e1e3
- 1.0*e1e2e3

Inner Product

The inner product is one of the three fundamental products of all geometric algebra, and it reduces the grade according to the constituent blades. Let $A$ and $B$ be blades of grades $k$ and $l$, respectively, then:

\[A \cdot B = \langle AB \rangle_{|k-l|}\]

It is applied between Blades and Multivectors and can be computed using the function inner_product or with the operator \.

julia> Algebra(3)

julia> e2 \ e1e2
- 1.0*e1

Outer Product

The outer product is the last of the three fundamental products of all geometric algebra, and it increases the grade according to the constituent blades. Let $A$ and $B$ be blades of grades $k$ and $l$, respectively, then:

\[A \wedge B = \langle AB \rangle_{k+l}\]

It is applied between Blades and Multivectors and can be computed using the function outer_product or with the operator ^.

julia> Algebra(3)

julia> e2^e1e3
- 1.0*e1e2e3

Scalar Product

The scalar product is defined as the scalar part of the geometric product.

\[A \ast B = \langle AB \rangle_{0}\]

It is applied between Blades and Multivectors and can be computed using the function scalar_product.

julia> Algebra(3)

julia> scalar_product(e1+e3+2, e1e2+1)
2.0

Left Contraction

The left contraction is defined as a product that produces a $(k-l)$-blade from a $k$-blade and a $l$-blade.

\[A \rfloor B = \langle AB \rangle_{k-l}\]

It is applied between Blades and Multivectors and can be computed using the function left_contraction or with the operator <<.

julia> Algebra(3)

julia> left_contraction(e1+e3+2, e1e2+1)
2 + 1.0*e2 + 2.0*e1e2

Right Contraction

The right contraction is defined as a product that produces a $(l-k)$-blade from a $k$-blade and a $l$-blade.

\[A \lfloor B = \langle AB \rangle_{l-k}\]

It is applied between Blades and Multivectors and can be computed using the function right_contraction or with the operator >>.

julia> Algebra(3)

julia> right_contraction(e1+e3+2, e1e2+1)
2 + 1.0*e1 + 1.0*e3

Regressive Product

The regressive product is quite similar to a meet (intersection). It is used to perform certain specific intersections, but it is not as general. Basically, the regressive product is the meet in which the join (union) is replaced by the pseudoscalar ($I$) (PERWASS).

\[A \nabla B = (A^* \wedge B^*)I\]

It is applied between Blades and Multivectors and can be computed using the function regressive_product or with the operator &.

julia> Algebra(3)

julia> (e1+e3+2) & (e1e2+eI)
3 + 1.0*e1 + 1.0*e3

Grade Functions

Grade

The grade of a blade is the number of elements from the Algebra that take part in the geometric product used to generate that blade. It is interesting to note that the grade of a scalar is 0, since no elements are multiplied to obtain it.

The grade of a multivector is the highest grade among all of its constituent blades. It can be computed using the function grade.

julia> Algebra(3)

julia> grade(e1e3)
2

julia> grade(1 + e2 + e3*e1*e2)
3

Grade Parts

There are additional grade functions that can be computed and used in various situations. This is the case for the functions grade plus, grade minus, and grade null, which return the grade of the multivector considering only the canonical blades whose geometric product is $+1$, $-1$ and $0$ respectively. Note that:

\[gr(A) = gr_+(A)+gr_-(A)+gr_0(A)\]

They can be computed using the functions grade_plus, grade_minus and grade_null.

julia> Algebra(2,2,2)

julia> grade_plus(Blade(e0e1e2e3e4e5))
2

julia> grade_minus(Blade(e0e1e2e3e4e5))
2

julia> grade_null(Blade(e0e1e2e3e4e5))
2

julia> grade(Blade(e0e1e2e3e4e5))
6

Grade Selection

Grade selection is a simple but fundamental operation for many other functions we defined. Thus, we say that the grade projection of a generic blade $A$ onto grade $k$ is:

\[\langle A \rangle_k = \begin{cases} A, & gr(A) = k \\ 0, & gr(A) \neq k \end{cases}\]

An interesting case is that grade selection can be generalized to multivectors, in such a way that it always selects all blades of grade $k$ from that multivector. It can be computed using the function grade_selection.

julia> Algebra(3)

julia> grade_selection(1+e1+e2+e1e3+e2e3, 2)
1.0*e1e3 + 1.0*e2e3

Involutions

Revert

The reverse operation is a unary operation performed on blades and is based on a very simple principle: reversing the order of all constituent elements, that is:

\[\widetilde{A} = \widetilde{(e_1e_2...e_k)} = (e_k...e_2e_1)\]

And by simplifying the expression, where $k$ is the grade of $A$, we arrive at:

\[\widetilde{A} = (-1)^{\frac{k(k-1)}{2}}A\]

It can be computed using the function revert. Or it can be used with the operator "~"

julia> Algebra(3)

julia> ~e1e2e3
- 1.0*e1e2e3

Conjugation

Conjugate

The conjugate operation is a unary operation performed on blades such that:

\[AA^\dagger = +1\]

Let $k$ be the grade of $A$ and $r$ be the grade_minus of $A$, then it can be written as:

\[A^\dagger = (-1)^r(-1)^{\frac{k(k-1)}{2}}A\]

It can be computed using the function conjugate. Notice that it only fulfills its role for non-degenerate algebras!

julia> Algebra(3)

julia> conjugate(e1e2e3)
- 1.0*e1e2e3

Clifford Conjugation

The clifford conjugation operation is a unary operation performed on blades for it's conjugate, similar to the above.

Let $k$ be the grade of $A$, then it can be written as:

\[\bar{A}= (-1)^{\frac{k(k+1)}{2}}A\]

It can be computed using the function clifford_conjugation. Notice that it only fulfills its role for non-degenerate algebras, but may have some applications.

julia> Algebra(3)

julia> clifford_conjugation(e1e2e3)
1.0*e1e2e3

Grade Involution

The grade involution operation is a unary operation performed on blades, it toggles the orientation of a blade if it's grade is odd.

Let $k$ be the grade of $A$, then it can be written as:

\[\hat{A}= (-1)^{k}A\]

It can be computed using the function grade_involution.

julia> Algebra(3)

julia> grade_involution(e1e2e3)
- 1.0*e1e2e3

Inversion

The inversion operation is a unary operation performed on blades such that:

\[AA^{-1} = +1\]

The inversion of generic multivectors is quite complicated, but rarely necessary. It is important to note that in degenerate algebras ($r \neq 0$) the nonexistence of an inverse can be common because the geometric product of some basis blades is zero.

\[A^{-1} = \frac{\widetilde{A}}{A\widetilde{A}}\]

It can be computed using the function invert. Notice that it only fulfills its role for non-degenerate algebras and mostly blades supported. You can also use the operator ^-1.

julia> Algebra(3)

julia> invert(e1+e2)
0.5*e1 + 0.5*e2

julia> e1e2e3^-1
- 1.0*e1e2e3

Dualization

The dual operation is a unary operation performed on blades or multivectors. It follows the principle of swapping the dimensions of a blade according to the signature. Note that dualization can be performed in any reference frame, but the pseudoscalar of the space $I$ is the one that gives it the most meaningful interpretation.

\[A^* = A \rfloor I^{-1}\]

Also, the undual, the inverse process, is defined as:

\[A^{-*} = A \rfloor I\]

It is interesting to note that dualization depends on the pseudoscalar being invertible, meaning that it does not work in this form for degenerate algebras. However, with the Hodge dual algorithm, we can also compute it in those cases, simply not using the equation above, but instead directly swapping the dimensions of the blade or multivector.

It can be computed using the function dual or undual. You can also use the operator ! for dual only.

julia> Algebra(3)

julia> dual(e1e3)
- 1.0*e2

julia> undual(-e2)
1.0*e1e3

julia> !(e1 + 2*e2 + 3*e3)
- 3.0*e1e2 + 2.0*e1e3 - 1.0*e2e3

Norms

A norm is a function that assigns a non-negative (or negative) real-valued ‘length’ to a multivector. There are many multiple norms that can be used.

It is related to (and used to calculate) the magnitude, that generalizes the familiar length of a vector to higher-grade geometric objects (like areas, volumes, etc.) and is the positive scalar value representing the "size" of any multivector.

A generic norm function that works both in degenerate and non-degenerate algebras called norm can be used.

julia> Algebra(2,2,1)

julia> norm(2*e1 + e2e3)
2.23606797749979

julia> norm(2*e0 + e2e3)
2.23606797749979

Reverse Norm

The reverse norm is a kind of norm defined as:

\[||A||_R^2 = \langle A \widetilde{A} \rangle_0\]

\[||A||_R = sign(||A||_R^2) \sqrt{|||A||_R^2|}\]

In practice the reverse norm is useful, especially due to its possible negative sign. It can be computed using the function reverse_norm.

julia> Algebra(2,2,1)

julia> reverse_norm(2*e1 + e2e3)
1.7320508075688772

julia> reverse_norm(2*e0 + e2e3)
-1.0

Euclidean Coefficients Norm

Let $A_k$ be the $k$-$th$ blade component of a multivector $A$, and let $S_k$ be the scalar of $A_k$, then the euclidean coefficients norm is a kind of norm defined as:

\[||M||_E = \sqrt{\sum_k S_{k}^2}\]

It is also good for degenerate algebras because the metric is mostly ignored. It can be computed using the function euclidean_norm.

julia> Algebra(2,2,1)

julia> euclidean_norm(2*e1 + e2e3)
2.23606797749979

julia> euclidean_norm(2*e0 + e2e3)
2.23606797749979

Conjugate Norm

The conjugate norm is a norm defined as:

\[||A||_\dagger = \sqrt{\langle AA^\dagger \rangle_0}\]

It can be computed using the function conjugate_norm.

julia> Algebra(2,2,1)

julia> conjugate_norm(2*e1 + e2e3)
2.23606797749979

julia> conjugate_norm(2*e0 + e2e3)
1.0

Clifford Norm

The clifford norm is a norm defined as:

\[||A||_C = \sqrt{|\langle A \bar{A} \rangle_0|}\]

It can be computed using the function clifford_norm.

julia> Algebra(2,2,1)

julia> clifford_norm(2*e1 + e2e3)
2.23606797749979

julia> clifford_norm(2*e0 + e2e3)
1.0

Is Scalar

The is_scalar is a function that checks where a GAType is a Scalar or not:

It can be computed using the function is_scalar.

julia> Algebra(3)

julia> is_scalar(1 + e1 - e1)
true

julia> is_scalar(e1)
false

Is Blade

The is_blade is a function that checks where a GAType is a Blade or not:

It can be computed using the function is_blade.

julia> Algebra(3)

julia> is_scalar(1 + e2)
false

julia> is_scalar(e1e2)
true

Exponential

The exponential is a function that computes the exponential of GA elements that squares to a scalar. A more generic function was not yet implemented.

It can be computed using the function exp_ga.

Algebra(2, 0, 1) # 2DPGA

# Rotate 'a' by an angle 'n' rad from the origin (0,0)
function rotate(a, n)
    rotor = exp_ga(-e1e2*n/2)
    return rotor * a * invert(rotor)
end

julia> exp_ga(-e1e2*pi/4)
0.7071067811865476 - 0.707*e1e2

Chain

The chain function 'chains' the canonical basis with a vector of coefficients. It sums the coeff[i]*canonical[i] with the rest, good for translating matrices lines into a GA object.

It can be computed using the function chain.

julia> Algebra(3)

julia> chain([1,2,3])
1.0*e1 + 2.0*e2 + 3.0*e3
