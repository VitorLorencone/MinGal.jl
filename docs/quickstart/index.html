Quick Start

To begin using the library for problem modeling, geometric structure representation, or any other mathematical application, the first step is always to define the algebra in which we will be working.

Tip - Algebra

An algebra Algebra(p, q, r) is defined by the following parameters:

  • p: Number of positive dimensions
  • q: Optional number of negative dimensions
  • r: Optional number of null dimensions

Let's create a 3D Euclidean Vectorspace:

julia> using MinGal
julia> Euclidean3D = Algebra(3)

Now, the necessary symbols have been automatically created for use throughout the rest of the code. In other words, we have at our disposal:

  • Grade-0 blade (Scalar): id
  • Grade-1 blades (Vectors): e1, e2, and e3
  • Grade-2 blades (Bivectors): e1e2, e1e3, and e2e3
  • Grade-3 blade (Trivector): e1e2e3 or eI

Among them, we can perform several operations in geometric algebra. Consider the main ones:

  • Product by Scalar: k * ei
  • Inner Product: ei \ ej
  • Outer Product: ei ^ ej
  • Geometric Product: ei * ej

Examples:

julia> (e1*e2 + e3)\eI
1.0*e1e2 - 1.0*e3

julia> e1^e1e3
0.0

julia> (5*e2 - 2*e2e3)/2
2.5*e2 - 1.0*e2e3

These operations produce objects known as Multivectors and can be stored and manipulated like any other data types.

Now that we know the basics, let's try to compute the area of a polygon in $R^2$ using geometric algebra concepts. For this, recall that in this algebra a point can be represented vectorially as $P(x,y) = x \circ e_1 + y \circ e_2$

point = (x, y) -> return x*e1 + y*e2

You can express the area of a polygon (ordered set $p$ of $n$ points) in geometric-algebra terms by triangulating the polygon with a fixed reference point (for example $p_1$) and summing the areas of the triangles. The area A is: $A = \frac{1}{2} \displaystyle \sum_{i=2}^{n-1} || (p_i-p_1) \wedge (p_{i+1}-p_1) ||$ where $||p||$ is the norm of a multivector or it's magnitude.

function area(points)
    area = 0
    p1 = points[1]
    for i in 2:length(points)-1
        area += (points[i]-p1)^(points[i+1]-p1)
    end

    return norm(area)/2
end

So, for an ordered set of points points:

julia> pts = [point(1,2), point(7.4, 2.3), point(4.17, -4.3), point(1.38, -1.12)]
julia> area(pts)24.3837

For now, you should already know the basics of the most commonly used functions and have seen a small example of what can be done, but there are still many other functions and possibilities, so I suggest taking a look at the next sections and examples to learn more!
