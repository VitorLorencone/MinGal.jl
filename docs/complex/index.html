Complex Plane

This example demonstrates a way to work with complex numbers using the Mingal library, in a user-friendly manner with regard to notation:

julia> Complex = Algebra(0, 1, 0, ["i"])
julia> i*i
- 1

julia> 5i^3
- 5.0*i

julia> (1+i)*(1+i)
2.0*i

Real Spaces

This example demonstrates a way to work with the R2 vectors using the Mingal library:

julia> R2 = Algebra(2,0,0,["i","j"])
julia> i^i
0.0

julia> i\j
0.0

julia> i^j
1.0*ij

julia> (i + 2*j)\(5*i+2*j)
9
